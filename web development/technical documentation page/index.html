Introduction

HTML (HyperText Markup Language) is the foundation of every website. It defines the structure and meaning of content on the web.

This guide will walk you through the basics of HTML so you can start building your own pages.

Basic Structure

Every HTML document follows a basic structure. Here is the minimum required:


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My First Page</title>
</head>
<body>
    <h1>Hello World!</h1>
</body>
</html>

The <head> contains metadata, while the <body> contains visible content.

Text Elements

HTML provides many tags for text formatting:

  • <h1>–<h6> headings
  • <p> paragraph
  • <strong> bold text
  • <em> italic text

<h2>This is a heading</h2>
<p>This is a paragraph.</p>
<strong>Bold text</strong>
<em>Italic text</em>

Links & Images

Links are created with the <a> tag, and images with the <img> tag.


<a href="https://example.com">Visit Example</a>

<img src="image.jpg" alt="A description of the image">

Lists

HTML supports ordered and unordered lists:


<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol>
    <li>First</li>
    <li>Second</li>
</ol>

Forms

Forms allow users to submit information:


<form>
    <label>Name:</label>
    <input type="text">

    <label>Email:</label>
    <input type="email">

    <button type="submit">Submit</button>
</form>

Conclusion

You now understand the basics of HTML. With these fundamentals, you can begin building real web pages and expand into CSS and JavaScript.
